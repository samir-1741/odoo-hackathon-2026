## ⚙️ How to Run

### Backend Setup
```bash
cd backend
pip install -r requirements.txt
python app.py
```
Server runs at: `http://localhost:5000`

### Frontend Setup
```bash
cd frontend
# Open index.html in browser directly
# OR use Live Server extension in VS Code
```
